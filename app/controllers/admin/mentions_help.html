Mentions: Guía rápida

Las mentions permiten insertar referencias a elementos del mundo (personajes, dones, objetos, etc.) dentro del texto de edición. Al escribir un prefijo con @, aparece un desplegable y puedes seleccionar el elemento deseado.

Uso básico

  • Escribe @tipo y luego el texto a buscar. Ejemplo: @character Ampw.
  • Selecciona con el ratón o con ↑/↓ y confirma con Enter.
  • En Quill se inserta un enlace directo.
  • En campos de texto plano se inserta un token con el formato [@tipo:id].
  • Al guardar, los tokens se convierten en enlaces con pretty_id.

En personajes se muestra también la Crónica en el desplegable.

Tipos disponibles

  • @character Personajes
  • @season Temporadas
  • @episode Episodios/Capítulos
  • @organization Organizaciones
  • @group Grupos
  • @gift Dones
  • @rite Ritos
  • @totem Tótems
  • @discipline Disciplinas
  • @item Objetos
  • @trait Rasgos
  • @background Trasfondos
  • @merit Méritos
  • @flaw Defectos
  • @merydef Méritos y Defectos
  • @doc Documentos
  • @system Sistemas
  • @breed Razas
  • @auspice Auspicios
  • @tribe Tribus

Ejemplos

  • @character Pedro → Inserta un enlace a Pedro Alba.
  • @gift Persuasión → Inserta el don correspondiente.
  • @item lanza → Inserta un objeto del inventario.

Notas

  • Si no aparecen resultados, revisa que el término exista en la base de datos.
  • El desplegable se cierra con Esc o al perder el foco.
